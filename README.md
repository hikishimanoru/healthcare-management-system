# HỆ THỐNG QUẢN LÝ DỊCH VỤ CHĂM SÓC SỨC KHỎE

## Giới thiệu

Đây là dự án xây dựng hệ thống quản lý dịch vụ chăm sóc sức khỏe bằng ngôn ngữ Python.

Hệ thống hỗ trợ quản lý:

- Bệnh nhân
- Bác sĩ
- Lịch khám
- Dịch vụ y tế

---

## Chức năng chính

- Đăng nhập hệ thống
- Quản lý bệnh nhân
- Quản lý bác sĩ
- Quản lý lịch khám
- Quản lý dịch vụ
- Báo cáo thống kê

---

## Công nghệ sử dụng

- Python
- Tkinter (thiết kế giao diện)
- SQLite (cơ sở dữ liệu)
- GitHub (quản lý mã nguồn)
